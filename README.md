# 🛡️ HypnotiXstudio Bot - Privacy Policy

The official, high-performance Privacy Policy landing page for the **HypnotiXstudio Telegram Bot**. This page is designed to provide a transparent and elegant overview of data handling practices for users.

![Project Status](https://img.shields.io/badge/Status-Live-success?style=for-the-badge)
![License](https://img.shields.io/badge/License-MIT-blue?style=for-the-badge)

## ✨ Features
* **Modern UI:** Glassmorphic design language with a dark tech aesthetic.
* **Responsive:** Fully optimized for mobile, tablet, and desktop viewing.
* **Performance:** Lightweight and fast-loading using Tailwind CSS.
* **Interactive:** Smooth scroll-reveal animations using vanilla JavaScript.
* **Compliance:** Clear sections covering data collection, storage, and user rights.

## 🛠️ Built With
* **HTML5** - Semantic structure.
* **Tailwind CSS** - Modern utility-first styling.
* **JavaScript** - Custom scroll-reveal logic.
* **Google Fonts** - Inter font family for high readability.

## 🚀 Deployment
This page is designed to be hosted via **GitHub Pages**.
1. Push the code to the `main` branch.
2. Navigate to **Settings > Pages**.
3. Select the `main` branch as the source and click **Save**.
